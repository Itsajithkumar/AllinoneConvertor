# All-in-One SaaS Product

## Overview
All-in-One is a comprehensive suite of tools designed to enhance productivity and streamline workflows. This project focuses on providing various utilities without a subscription model, making it accessible to all users.

## Features
- **Image to Text Converter**: Convert images into editable text using advanced OCR technology.
- **Image Translator**: Translate text within images to different languages seamlessly.
- **User-Friendly Interface**: Intuitive design for easy navigation and tool access.
- **No Subscription Required**: Enjoy all features without the need for a subscription.

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/all-in-one.git
   ```
2. Navigate to the project directory:
   ```
   cd all-in-one
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Application
To start the development server, run:
```
npm run dev
```
Open your browser and navigate to `http://localhost:3000` to view the application.
